# Summary

SGLang is an AI inference serving and model runtime project. In this domain it belongs with serving infrastructure because the captured issue and pull-request corpus centers on runtime behavior, distributed serving, model/backend support, performance work, kernel integration, memory and KV-cache behavior, and reliability.

# Operational Evidence

The local raw corpus captures closed GitHub issue and pull-request history for `sgl-project/sglang` from January 2024 through June 24, 2026. It includes 5,633 closed issues, 19,475 closed pull requests, 76,373 attached issue/PR timeline comments, and 46,819 attached pull-request review comments.

The closed pull-request set contains 14,670 merged PRs and 4,805 closed-unmerged PRs, so closed PRs should not be interpreted as merged changes without checking `merged_at`.
