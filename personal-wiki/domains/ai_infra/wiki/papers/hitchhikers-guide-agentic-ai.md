# Summary

Haggai Roitman's 2026 arXiv guide is a broad book-length reference on agentic AI, from transformer and training foundations through AI systems, evaluation, and deployment of autonomous agents. In the `ai_infra` domain, its durable value is as a map of infrastructure-facing topics rather than as a source of new accelerator specifications.

# Infrastructure-Relevant Coverage

- Systems foundations include GPU architecture, memory hierarchy, parallelism, inference optimization, distributed training, and production deployment concepts.
- Evaluation coverage distinguishes normal LLM evaluation from agentic evaluation, where multi-step task success, reproducibility, benchmark integrity, and environment state matter.
- The agentic stack is organized around memory systems, harness/orchestration, RAG, context management, tool integration, environments and benchmarks, MCP, A2A, multi-agent coordination, frameworks, and agentic UI.
- MCP is treated as a standardized client-server interface for connecting agent harnesses to tools, resources, and prompts; A2A is treated as the complementary interface for inter-agent communication.

# Use In This Wiki

Use this raw source as a reference when organizing pages about agent harnesses, MCP/A2A integration, agent evaluation environments, production orchestration, and LLM systems foundations. Do not treat it as primary evidence for vendor hardware parameters or resolved accelerator catalog fields.
