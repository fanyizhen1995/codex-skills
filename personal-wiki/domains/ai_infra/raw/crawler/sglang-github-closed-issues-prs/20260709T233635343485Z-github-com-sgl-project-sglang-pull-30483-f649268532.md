# Enhance mechanical refactor proof construction and verification skill

URL: https://github.com/sgl-project/sglang/pull/30483

Closed at: 2026-07-08T23:39:11Z
Merged at: 2026-07-08T23:39:11Z

Stacked on #28978 (base branch = `tom/refactor_agent_skill`); merge that one first.

Makes mechanical refactors machine-checkable: one property (*a commit is a pure relocation*), one proof (**reproduce** — regenerate the move from the base commit with faithful relocation primitives, run the formatter, byte-diff against the target).

- `scripts/mechanical_refactor_proof_generator.py` — infers a reproduce recipe from a commit's diff, emits/runs a standalone auditable script per commit with a PASS / RESIDUAL / UNSUPPORTED verdict (range mode writes a self-contained folder with log + HTML report).
- `scripts/mechanical_refactor_reproduction_utils.py` — the proof engine: the `Repro` builder's relocation primitives (move/extract/call-site/import ops) plus the worktree + pre-commit + byte-diff arbiter. Primitives splice original source text (never `ast.unparse`), audit authored surfaces (new-module headers, delegate stubs), and fail loudly on anything non-mechanical.
- Docs: `spec-reproduction-utils.md` (source of truth for the property and primitive contracts) + three role guides (`guide-split.md`, `guide-construct-proof.md`, `guide-verify-proof.md`), deduplicated and section-numbered.
- Tests: `scripts/tests/`, one folder per module, one file per primitive — 130 passed locally (`uv run --with pytest pytest tests/ -q`, Python 3.14.4), including byte-exact adversarial edge cases (decorators nested in moved bodies, semicolon-joined imports, CRLF, UTF-8 column offsets, magic trailing commas, chained assigns, string-literal interiors).

Many conventions here follow @merrymercy's style guidance.
