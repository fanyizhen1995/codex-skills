## Accuracy Tests

<!-- If this pull request affects model outputs (e.g., changes to the kernel or model forward code), provide accuracy test results. -->
`test/registered/sampling/test_sampling_mask.py` was added to check the following:
- basic validations: each sampled token has an attached sampling mask, size of each sampling mask <= specified top-k, and the sampled token always exists in the sampling mask.
- verifies under deterministic inference that sampled tokens when `return_sampling_mask=True` are identical to those returned when `return_sampling_mask=False`, the default path.
- validates that the sampling mask and renormalized logprobs are mathematically correct. This is done by querying the server with `return_logprob = True` and `top_logprobs_num = TOP_K` with temperature 1, which returns the unnormalized logprobs of the TOP_K most probable tokens. From here, we can reconstruct the sampling mask (modulo some possible boundary off-by-ones) and compute the renormalized logprobs offline.
