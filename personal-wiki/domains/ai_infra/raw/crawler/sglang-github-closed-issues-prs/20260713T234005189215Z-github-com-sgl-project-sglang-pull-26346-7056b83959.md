# Add mooncake_tcp transfer backend (mooncake over TCP)

URL: https://github.com/sgl-project/sglang/pull/26346
State: closed
Labels: 
Closed at: 2026-05-27T01:15:56Z
Merged at: 2026-05-27T01:15:56Z

## What
Adds `--disaggregation-transfer-backend mooncake_tcp`, a thin alias that runs the existing mooncake backend over TCP instead of RDMA. It sets `MC_FORCE_TCP=1` (so mooncake installs TcpTransport), rewrites the backend to `mooncake`, and skips RDMA HCA selection.

## Why
Provides a no-RDMA path for PD disaggregation where RDMA is unavailable or unstable, without duplicating any mooncake bootstrap / transfer / staging logic.

## Notes
- `MC_FORCE_TCP` is read internally by the mooncake transfer engine; requires a mooncake build that honors it.
- The alias rewrite is logged at startup so the effective backend stays visible.
