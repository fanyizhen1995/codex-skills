## Modifications

- Add LingBot realtime support for composite prompt and camera-action inputs in a single event.
- Add prompt-update handling that resets LingBot cross-attention cache only when the prompt changes.
- Add optional LingBot interactive KV window control, disabled by default and enabled through `SGLANG_LINGBOT_ENABLE_INTERACTIVE_KV_WINDOW`.
  - Dynamically adjusts LingBot realtime KV sampling windows for static vs. moving camera-control chunks, improving motion continuity while keeping the default path unchanged.
- Add optional LingBot lazy VAE encode control, disabled by default and configured through `SGLANG_LINGBOT_LAZY_VAE_ENCODE_BLACK_FRAMES`.
  - Encodes only the initial image plus a configurable number of black padding frames, then extends the latent condition to avoid unnecessary VAE work on long padding tails.
