## Modifications
Adds a registered config variant for `robbyant/lingbot-world-v2-diffusers`.

This checkpoint uses the same LingBot-World causal DMD pipeline as the existing fast diffusers checkpoint, with different default inference parameters:
- `dmd_denoising_steps = [1000, 750, 500, 250]`
- `flow_shift = 5.0`
