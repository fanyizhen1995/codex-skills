# Deprecate and delete `--moe-runner-backend cutlass`, dense FP4 GEMM, FP4 quantization (in sgl-kernel)

This kernel was originally ported from TRTLLM. And ever since it's integration in Flashinfer, we mainly call these APIs. Since it doesn't offer any special usages, we could delete it (as the Flashinfer version is more maintained).

We might also consider deleting the dense FP4 GEMM too, and FP4 qantization. As it doesn't offer any special usages
