## Summary
- Replace several broad per-commit JIT kernel test grids with small frozen representative shape tables while keeping broader local/nightly coverage through `get_ci_test_range`.
- Add representative production-like shapes for RMSNorm, fused-add RMSNorm, activation, FP8 quantization, per-token group quantization, router GEMM, fused-A GEMM, and MoE align.
- Add dedicated `sgl-kernel` correctness cases for FP8 scaled-mm, per-token FP8 quantization, norm/fused norm, and fused-A GEMM.

Part of the test-hardening baseline for RFC #29630. This also gives future agent-kernel work a clearer correctness baseline before kernel APIs move around.
