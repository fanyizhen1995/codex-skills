## Summary

Extends the AMD MI355X 2-node 1P1D MORI disaggregation nightly with extra topology and speculative-decode coverage for all four DeepSeek-V4 model x precision combos (dsv4flash/dsv4pro x fp8/fp4). Three new variants per model (12 recipes + 12 config entries):

- `1p1d-mtp`: TP8 + EAGLE MTP
- `1p1d-dp8ep8`: DP-attention 8 + narrow within-node EP8
- `1p1d-dp8ep8-mtp`: DP8 + narrow EP8 + EAGLE MTP

`launch_mi355x.sh` now reads `expert-parallel-size` / `data-parallel-size` and an optional `mtp:` recipe section, appending `--ep-size` / `--enable-dp-attention --dp-size` / `--speculative-*` flags. Base recipes (EP1/DP1, no mtp) yield empty extra flags, so prior behavior is preserved. The existing base `1p1d.yaml` recipes are unchanged.
